# 诚书记 TTS 服务端启动命令

## SoulX-Podcast（默认推荐）

```bash
python -m chengbook_tts.cli serve --model soulxpodcast --host 0.0.0.0 --port 18080
```

访问地址：`http://localhost:8080/` 或 `http://192.168.80.74:8080/`

## CosyVoice3

```bash
python -m chengbook_tts.cli serve --model cosyvoice3 --host 0.0.0.0 --port 18080
```

## IndexTTS v1/v1.5

```bash
python -m chengbook_tts.cli serve --model indextts_v1 --host 0.0.0.0 --port 18080
```

## IndexTTS v2

```bash
python -m chengbook_tts.cli serve --model indextts_v2 --host 0.0.0.0 --port 18080
```

## 切换模型

```bash
python -m chengbook_tts.cli model list           # 查看可用模型
python -m chengbook_tts.cli model info <name>     # 查看模型详情
python -m chengbook_tts.cli model switch <name>   # 切换模型
```

## 直接合成

```bash
python -m chengbook_tts.cli synth --text "你好" --voice woman --model soulxpodcast --output output.wav
```

## 端口说明

服务内部使用 18080 端口，外部通过 portproxy 转发 8080 → 18080，重启服务不影响外部访问。

重新配置 portproxy（管理员 PowerShell）：

```powershell
netsh interface portproxy delete v4tov4 listenport=8080 listenaddress=0.0.0.0
netsh interface portproxy add v4tov4 listenport=8080 listenaddress=0.0.0.0 connectport=18080 connectaddress=127.0.0.1
```
